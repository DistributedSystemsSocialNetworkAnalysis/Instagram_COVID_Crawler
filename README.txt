# Instagram_COVID_Crawler
#


** GENERAL INFORMATION **
This code has been developed in order to study how information concerning the pandemic affected the Instagram Social Media. It represents a Instagram crawler, developed by using Selenium.
The crawler collect only PUBLIC INFORMATION containing tags related to the pandemic.
